## 🚀 Experience the Precision

The **Khmer Weather App** by **rikixz** is more than just a weather forecast. It's an engineered data dashboard that brings real-time atmospheric insights to your fingertips with a "man-built" industrial aesthetic.

### Key Features
- **Engineered UI**: Clean, structured layout with a layered glassmorphism design system.
- **Micro-Metric Precision**: Real-time tracking of Temperature, Humidity, Wind Speed, Pressure, and Visibility.
- **Global Reach**: Instant insights for any city or region worldwide via the `wttr.in` API.
- **Custom Vector Graphics**: Hand-crafted SVG icons for all weather states — zero emojis, just pure design.
- **Deployment Ready**: Fully optimized for Vercel and GitHub with a lean codebase.

## 📦 Installation & Setup

Get the professional environment running locally in seconds:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/rikixz/khmer-weather-app.git
   cd khmer-weather-app
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Configure Environment**:
   Create a `.env` file in the root directory:
   ```env
   VITE_WEATHER_API_URL=https://wttr.in/
   ```

4. **Launch Developer Console**:
   ```bash
   npm run dev
   ```
